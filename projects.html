SpotiRecommend

This application takes an input song from the user and generates a list of 20 songs that are matched in similarity, sorting them based on the user's recommendation.

This algorithm was made using the Spotify API​ and Python, and is currently open to test on my GitHub.

RUN!

​This game is a endless runner zombie game from a top down perspective. As you find yourself deep in the heart of a never ending zombie apocalypse, you must continue to run, find shelter, and survive as long as possible.

This game is being made in the Godot Engine utilizing Python and the PyGame libraries.
